## Curso Automacão Rest Assured do WC Aquino
Este é um repositorio que contém código da automação de testes de um software chamado Seu Barriga. Os sub-tópicos abaixo descrevem algumas decições tomadas na estruturação do projeto.

### Tecnologias Utilizadas

 - Java
 - RestAssured
 - JUnit
 - Maven

### Notas Gerais

 - Utiliza @beforeClass para indicar os metodos de iniciação
 - Utiliza @FixMethodOrder para ordenar os testes
 - Utiliza Matchers para realizar validações
